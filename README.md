# Crypto Monitor

一个实时加密货币监控仪表板,支持比特币和以太坊的价格追踪、图表分析和AI投资建议。

## 🌟 功能特性

- **实时价格更新**: 通过 Binance WebSocket 实时获取 BTC 和 ETH 价格
- **多时间段统计**: 显示 1小时、4小时、24小时 的价格变化
- **交互式图表**: 使用 Chart.js 展示 24小时价格走势
- **High/Low 标记**: 自动标注图表中的最高价和最低价
- **AI 投资建议**: 基于市场趋势提供中文投资建议
- **汇率计算器**: 实时 USD/CNY 汇率转换
- **响应式设计**: 适配各种屏幕尺寸

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **图表库**: Chart.js
- **数据源**: Binance API & WebSocket
- **汇率API**: ExchangeRate-API
- **字体**: Google Fonts (Outfit)

## 📦 本地运行

1. 克隆仓库:
```bash
git clone https://github.com/muyi.66@163.com/crypto-monitor.git
cd crypto-monitor
```

2. 直接打开 `index.html` 文件即可使用(无需构建步骤)
